## Why Ironflow

Ironflow turns raw operational CSV data into validated JSON output with deterministic ordering and predictable runtime.

- Parallel transforms via Rayon
- CI-gated quality checks + coverage threshold
- Docker image and Kubernetes job included

## Outcome

Use Ironflow when you need data transformations that run identically in local dev, CI, and cluster jobs.
